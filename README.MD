# Setup environment

In the project root run:
```
virtualenv env
source env/bin/activate
pip install -r requirements.txt
```


# Run the server

In the project root run:

```
python src/main.py
```


# Running tests

In the project root run:

```
 python -m unittest discover src
```

or 

Inside the folder src/

```
 python -m unittest discover
```
